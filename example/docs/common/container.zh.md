#### width

<description>**可选** _number_</description>

功能描述： 设置图表宽度。

默认配置： `400`

#### height

**可选**, _number_

功能描述： 设置图表高度。

默认配置： `400`

#### forceFit

**可选**, _boolean_

功能描述： 图表是否自适应容器宽高。当 `forceFit` 设置为 true 时，`width` 和 `height` 的设置将失效。

默认配置： `true`
